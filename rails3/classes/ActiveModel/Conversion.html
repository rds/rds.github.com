Module ActiveModel::Conversion

  1. vendor/gems/gems/activemodel-3.0.0.beta/lib/active_model/conversion.rb

If your object is already designed to implement all of the Active Model featurs include this module in your Class. 

class MyClass
  include ActiveModel::Conversion
end

Returns self to the :to_model method.

If your model does not act like an Active Model object, then you should define :to_model yourself returning a proxy object that wraps your object with Active Model compliant methods.
